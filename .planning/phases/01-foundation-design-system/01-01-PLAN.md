  <name>Task 2: Create design token system, base typography, and test page</name>
  <files>src/styles/global.css, src/components/BaseHead.astro, src/pages/index.astro</files>
  <action>
    **Create `src/styles/global.css`** with these clearly sectioned blocks (single file, ~150-200 lines):

    1. **CSS Reset** -- Modern minimal reset (box-sizing border-box on all elements, margin 0, text-size-adjust, body line-height 1.6 + font-smoothing, block display on media elements, font inherit on form elements, overflow-wrap break-word on headings/paragraphs).

    2. **Design Tokens** on `:root`:
       - Font families: `--font-heading: 'Space Grotesk Variable', system-ui, sans-serif` and `--font-body: 'Inter Variable', system-ui, sans-serif`
       - Fluid type scale using `clamp()` with `rem + vw` in preferred values (NOT pure vw -- must respect user font preferences). Use a 1.25 ratio: `--text-sm`, `--text-base`, `--text-lg`, `--text-xl`, `--text-2xl`, `--text-3xl`. Use the exact values from the research: e.g. `--text-base: clamp(1rem, 0.9rem + 0.5vw, 1.125rem)`, `--text-3xl: clamp(2.5rem, 1.5rem + 5vw, 5rem)`
       - Fluid spacing scale with `clamp()`: `--space-xs` through `--space-2xl`
       - Colors: `--color-bg: #FAFAFA`, `--color-text: #1A1A1A`, `--color-text-muted: #6B6B6B`, `--color-accent: #2563EB`, `--color-accent-hover: #1D4FD8`, `--color-border: #E5E5E5`
       - Layout: `--max-width: 72rem`, `--content-width: 42rem`

    3. **Base Typography**: body uses `--font-body` and `--text-base`. Headings (h1-h4) use `--font-heading`, weight 700, line-height 1.1, letter-spacing -0.02em. Map h1 to `--text-3xl`, h2 to `--text-2xl`, h3 to `--text-xl`, h4 to `--text-lg`.

    4. **Layout Primitives**: `.container` (max-width + auto margins + inline padding), `.narrow` (content-width), `.stack` and `.stack-lg` (vertical spacing via lobotomized owl), `.section` (block padding with `--space-2xl`).

    5. **Utility classes**: `.text-lg` and `.text-sm` for overriding font size.

    Do NOT add animation classes yet -- those come in Plan 02.

    **Create `src/components/BaseHead.astro`** that imports the two Fontsource variable font packages and global.css. Accept `title` and `description` props with sensible defaults ("Zero Fluff - AI Implementation That Works" and "Straight-talking AI consultancy for small businesses. No jargon, no fluff, just results."). Include charset, viewport, description meta, title tag, and favicon link.

    **IMPORTANT (FOUN-02): Add a font preload hint** for the Space Grotesk Variable WOFF2 file. After importing `@fontsource-variable/space-grotesk`, resolve the WOFF2 path and add a preload link in the `<head>`. In the Astro frontmatter, import the WOFF2 file directly to get its resolved URL:
    ```astro
    ---
    import spaceGroteskWoff2 from '@fontsource-variable/space-grotesk/files/space-grotesk-latin-wght-normal.woff2?url';
    ---
    ```
    Then in the template, before other link tags, add:
    ```html
    <link rel="preload" href={spaceGroteskWoff2} as="font" type="font/woff2" crossorigin />
    ```
    This ensures the primary heading font begins downloading immediately rather than waiting for CSS parsing. Only preload Space Grotesk (headings) -- Inter (body) can load normally as it is less visually jarring if it swaps in slightly later.

    **Replace `src/pages/index.astro`** with a test page that proves the design system works. It should import BaseLayout (if it exists) or use BaseHead directly, and render:
    - A hero section with an h1, subtitle paragraph, and a styled button/link
    - A typography showcase section showing all heading levels (h1-h4) and body text sizes
    - A spacing/layout demo section using `.container`, `.narrow`, `.stack` classes
    - Multiple sections to test scroll (even though animations come later, having scrollable content is needed)

    Since BaseLayout doesn't exist yet (Plan 02), have index.astro use BaseHead directly in a basic HTML structure:
    ```astro
    ---
    import BaseHead from '../components/BaseHead.astro';
    ---
    <html lang="en">
      <head><BaseHead /></head>
      <body>
        <!-- sections here -->
      </body>
    </html>
    ```

    Ensure all spacing and sizing uses CSS custom properties, not hardcoded values. The page should feel spacious and typography-led even without animations or navigation.
