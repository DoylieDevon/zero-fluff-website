# Zero Fluff Website

## What This Is

A redesigned website for Zero Fluff — Andy Doyle's AI implementation consultancy based in Totnes, Devon. The site replaces the current generic template at zerofluff.co.uk with a typography-led, minimal design that reflects the straight-talking, no-nonsense brand. It's a static site built to earn trust first and convert visitors to enquiries second.

## Core Value

Visitors immediately understand what Zero Fluff does (practical AI implementation for businesses) and trust Andy enough to get in touch — through confident design, real case studies, and clear, direct messaging.

## Context

- Current site is a template-based design at zerofluff.co.uk that feels generic and doesn't reflect Andy's personality
- Zero Fluff is an AI/automation consultancy helping small businesses implement practical AI solutions
- Andy has 30+ years digital product experience, previously built Filmily, SelectCommerce, and Shiift
- Uses tools like N8N, ChatGPT, Gemini in client work
- The brand name "Zero Fluff" should be reflected in the design — minimal, direct, no unnecessary decoration
- Inspiration site (physical-light-947386.framer.app) demonstrates the desired aesthetic: whitespace, big confident typography, neutral palette
- Andy already writes on Substack — blog pulls from there rather than maintaining separate content
- Current services (Quick Start, Automation Setup, Ongoing Support, Strategic Consultation) to be simplified into a clearer, more focused offering

## Constraints

- **Tech stack**: Static HTML/CSS/JS — no frameworks unless they add clear value
- **Hosting**: Vercel free tier
- **Domain**: zerofluff.co.uk (existing)
- **Blog**: Substack as content source — embed or link, no separate CMS
- **Design**: Typography-led, generous whitespace, minimal color palette with one bold accent
- **Tone**: Straight-talking, direct, no corporate jargon — the brand IS zero fluff
